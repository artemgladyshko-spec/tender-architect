# User Feedback Patterns

Interfaces must clearly communicate system feedback.

---

# Feedback Types

Inline feedback

validation messages  
field hints

Global feedback

notifications  
alerts  
status banners

---

# Notifications

Use short messages to inform users about actions.

Examples:

"File uploaded successfully"

"Analysis started"

---

# Alerts

Alerts should be used for critical events.

Examples:

system error  
action failure

Alerts should include clear resolution guidance.

---

# Confirmation Patterns

Destructive actions should require confirmation.

Examples:

Delete document  
Cancel analysis
